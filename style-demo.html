Vue d'ensemble

ClassCord est un client de messagerie instantanee developpe en Java Swing, concu pour permettre aux professeurs et etudiants de communiquer en temps reel sur le reseau local de l'etablissement. Le client se connecte a un serveur de chat existant via une adresse IP et un port configurables.

Captures d'ecran

Fonctionnalites principales

  • Connexion a un serveur distant via IP et port configurables
  • Trois modes de connexion : invite, connexion rapide, compte utilisateur
  • Affichage des messages du canal global en temps reel
  • Envoi et reception de messages prives
  • Gestion des etats utilisateurs (disponible, absent, invisible)

Defis techniques

Programmation Reseau en Java

Implementation de communications TCP/IP avec gestion des sockets et threads reseau separes pour la reception asynchrone.

Architecture MVC Sans Framework

Conception d'une architecture MVC personnalisee implementee manuellement, separant Model, View et Controller.

Resultats

Competences Cles Mobilisees

Maitrise de la modelisation, architecture logicielle, programmation reseau et interface graphique Java Swing.

Methodologie Iterative

Projet complet realise en 5 jours avec planification, developpement progressif et deploiement fonctionnel.

Vue d'ensemble

ClassCord est un client de messagerie instantanee developpe en Java Swing, concu pour permettre aux professeurs et etudiants de communiquer en temps reel sur le reseau local de l'etablissement. Le client se connecte a un serveur de chat existant via une adresse IP et un port configurables, offrant une interface fluide et reactive.

Le projet mobilise les competences en modelisation, architecture logicielle, programmation reseau et interface graphique, suivant une methodologie de projet iterative par etapes.
