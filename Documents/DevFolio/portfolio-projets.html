Chat Interactif sur Serveur Dédié

Application de chat en temps réel développée en Java (architecture MVC) avec interface graphique Swing. Permet d’échanger des messages en public ou en privé, de gérer les états de présence (en ligne, absent, ne pas déranger, invisible) et de se connecter via plusieurs modes (inscription, connexion rapide, invité). Hébergée sur un serveur dédié.

Java MVC Swing Socket Serveur dédié

Mini Jeu de Bataille sur Arène

Jeu web multijoueur où chaque joueur contrôle un personnage sur une arène et affronte d'autres joueurs en temps réel. Réalisé en JavaScript (Node.js, Express, Socket.io) avec une interface graphique dynamique et un système de classement.

JavaScript Node.js Express Socket.io

ENT Prof/Élève – Gestion de Notes

Application de gestion scolaire développée en Java avec base de données interne, permettant aux élèves de consulter leurs notes, moyennes et commentaires après connexion sécurisée (nom d’utilisateur et mot de passe). Les professeurs disposent d’une interface dédiée pour ajouter des matières, saisir des notes et commentaires, et consulter le détail des résultats de chaque élève.

Java Base de données Swing Gestion scolaire
